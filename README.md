## SC2

A small sc2 API wrapper. For full documentation, visit https://github.com/Blizzard/api-sc2-docs.

## Installation

Add to your `Gemfile`:

```ruby
gem "sc2"
```

Then `bundle install`.

## Usage

```ruby
SC2::Client.new(host: 'us.battle.net')
```
